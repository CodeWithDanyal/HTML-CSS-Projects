# Profile Card UI Design

## Description
This is a simple and elegant profile card UI design created using HTML and CSS. The card showcases a profile picture, name, bio, social links, and action buttons, making it suitable for a personal website or portfolio.

## Features
- Profile image with a circular border
- Name and bio section
- Social media links with icons
- Action buttons for subscribing and messaging
- Display of followers statistics

## Technologies Used
- HTML
- CSS
- Boxicons (for social media icons)
- Google Fonts (Poppins)

## Usage
To use this profile card in your project:
1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Customize the content in the HTML and CSS files as per your requirement.
